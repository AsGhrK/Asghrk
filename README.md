## Sobre mim

Nascido em 2006, sou de Nova Andradina (MS) e estou cursando **Análise e Desenvolvimento de Sistemas** no IFMS. Gosto de construir coisas que funcionam de verdade — de bibliotecas de livros técnicos a projetos de código aberto que facilitam a vida de outros devs.

- Estudante de ADS — IFMS (2025 - 2028)
- Focado em desenvolvimento de software
- Contribuidor open source
- Mantenedor de biblioteca técnica open source
